1. Introduction

In this post we talk about the inverse matrix. We’ll talk about what it means for a matrix to be invertible, for it not to be invertible (singular) and how to find a matrix’s inverse.

2. Invertibility

Let’s think about a square matrix, $A$. In a previous post we spoke about how a matrix acts on a vector. The inverse of $A$ is $A^{-1}$. It undoes what $A$ did.

If we have a vector $x$, and we multiply it by our matrix $A$, we get $Ax$. If we undo this operation by applying the inverse, we get back to $x = A^{-1}Ax$. The one matrix which always leaves any vector unchanged is the identity matrix, $I$; which is why $A^{-1}A = I$.

When matrix $A$ has an inverse it is said to be invertible. However, this is not always the case. When a matrix is not invertible, it is also known as singular.
